# Comparative Analysis of Naive, Rabin-Karp and KMP Algorithms

## Aim
To implement and compare Naive Search, Rabin-Karp and Knuth-Morris-Pratt (KMP) algorithms for string matching.

## Features
- Naive Search
- Rabin-Karp Algorithm
- KMP Algorithm
- Comparison of character comparisons
- Performance Analysis

## Technologies Used
- Python 3

## Time Complexity

Naive Search
Worst Case : O(nm)

KMP
Worst Case : O(n+m)

Rabin-Karp
Average Case : O(n+m)
